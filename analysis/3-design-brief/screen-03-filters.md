Экран: Шторка фильтров
Связанный файл: `design-brief.md`
UC: UC-2 (Просмотр и фильтрация списка слотов)
Порядковый номер: 3

Цель
Быстро отфильтровать слоты.

Связь с требованиями
FR: FR-38
US: US-3
UC: UC-2

Структура экрана
Период: сегмент-контрол «Сегодня / Эта неделя / Выходные / Кастомный» + date-picker для кастомного.
Формат: чипы «Новички» / «Опытные» (мультивыбор, OR внутри группы).
Инструктор: список чипов с именами (скрывается, если справочник пуст — A4 UC-2).
Опция: «Только свободные» (toggle).
Кнопки «Сбросить» и «Применить».

Состояния
| Состояние | Поведение |
|---|---|
| Default | Все фильтры сброшены, «Применить» disabled (если нет изменений) |
| Выбрано | «Применить» active, счётчик применённых фильтров |
| Offline | Модалка недоступна, баннер «Нет соединения» |
| Пустой справочник инструкторов | Блок «Инструктор» скрыт (A4 UC-2) |

Заметки для дизайнера
- Модалка фильтров: на мобильных — bottom sheet (шторка снизу), на десктопе — dialog/dropdown по центру.
- Семантика фильтров: внутри одной группы значения комбинируются по ИЛИ (формат, инструктор — массивы), между группами — по И. Границы периода дат включительные (FR-38).
- «Применить» перезагружает ленту с анимацией skeleton.
- «Сбросить» возвращает к дефолтной выдаче (FR-9).

Экран: Шторка фильтров

Связанный файл: `design-brief.md`
UC: UC-2 (Просмотр и фильтрация списка слотов)
Порядковый номер: 3

## Цель

Быстро отфильтровать слоты.

## Связь с требованиями

- FR: FR-38
- US: US-3
- UC: UC-2

## Структура экрана

- **Период:** сегмент-контрол «Сегодня / Эта неделя / Выходные / Кастомный» + date-picker для кастомного.
- **Формат:** чипы «Новички» / «Опытные» (мультивыбор, OR внутри группы).
- **Инструктор:** список чипов с именами (скрывается, если справочник пуст — A4 UC-2).
- **Опция:** «Только свободные» (toggle).
- **Кнопки** «Сбросить» и «Применить».

## Семантика фильтров (FR-38)

- Внутри группы — OR (например, выбраны и новичковый, и опытный формат → показать оба).
- Между группами — AND (например, формат И период дат И инструктор).
- Границы периода дат — включительные.
- Даты вводятся в локальной дате клиента.
- По умолчанию фильтры не заданы (дефолтная выдача — 7 дней).
- Если справочник инструкторов пуст — фильтр по инструктору скрывается (A4 UC-2).

## Состояния экрана

| Состояние | Поведение |
|---|---|
| Пустые фильтры | Все значения по умолчанию, кнопка «Сбросить» disabled |
| Есть активные фильтры | Кнопка «Сбросить» активна |
| Применение | Закрытие модалки, обновление ленты (screen-02-schedule.md) |

## Заметки для дизайнера

- **На мобильном:** модалка открывается снизу (bottom sheet).
- **На десктопе:** модалка открывается по центру (dialog) или как dropdown.
- Активные фильтры отображаются чипами на основном экране.
- Кастомный диапазон дат — date-picker с двумя полями «с / по».

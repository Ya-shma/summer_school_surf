Экран: Оценка инструктора
Связанный файл: `design-brief.md`
UC: UC-6 (Оценка инструктора)
Порядковый номер: 8

Цель
Быстро оценить после тренировки.

Связь с требованиями
FR: FR-57, FR-58
US: US-17
UC: UC-6

Структура экрана
Заголовок: «Как прошла тренировка?»
Инструктор: имя, дата тренировки.
5 звёзд (тап/клик по звезде — выбор).
CTA «Отправить оценку» (активна после выбора).
Успех → тост «Спасибо за оценку!» → возврат в «Мои бронирования».

Состояния экрана
| Состояние | Поведение |
|---|---|
| Нет оценки | 5 пустых звёзд, CTA «Отправить оценку» disabled |
| Выбрана оценка | Звёзды заполнены, CTA активна |
| Уже оценено | Экран показывает уже выбранное количество звёзд и позволяет изменить оценку. CTA называется «Изменить оценку» |
| Загрузка | Loading-индикатор на CTA |
| Успех | Тост «Спасибо за оценку!» |

Матрица ошибок
| Код | HTTP | UX-реакция |
|---|---|---|
| not_attended | 403 | Оценка не принимается, показывается сообщение |
| rating_window_closed | — | «Окно оценки закрыто» |
| Нет сети | — | Retry с тем же `Idempotency-Key` (NFR-27) |

Ограничения
Нет текстовых комментариев (FR-58).
Оценка видна только владельцу (не показываем клиенту «средний рейтинг»).
Доступно только для статуса `attended` (FR-57).
Push приходит через 1 час после окончания тренировки (FR-57).

Заметки для дизайнера
Deep-link из push ведёт прямо на этот экран.
Звёзды — крупные, удобные для тапа/клика (≥ 44px).
После отправки — возврат в «Мои бронирования».

Экран: Редактирование профиля
Связанный файл: `design-brief.md`
UC: UC-7 (Управление профилем и выход из аккаунта)
Порядковый номер: 10

Цель
Обновить данные.

Связь с требованиями
FR: FR-47
US: US-19
UC: UC-7

Поля
Имя (обязательно).
Телефон (только просмотр, смена — через Telegram + ручная привязка владельцем).
Возраст (опционально).
День рождения (опционально).

CTA
«Сохранить» → при успехе тост «Данные обновлены».

Состояния экрана
| Состояние | Поведение |
|---|---|
| Загрузка | Skeleton |
| Данные | Поля заполнены |
| Изменения | CTA «Сохранить» активна |
| Успех | Тост «Данные обновлены» |
| Ошибка | Тост «Не удалось сохранить» + Retry |

Заметки для дизайнера
Телефон — read-only, смена только через владельца (A1 UC-7).
Возраст и день рождения — опциональные поля.
Валидация имени — обязательно.
Форма доступна в веб-приложении на мобильном и десктопе (responsive).

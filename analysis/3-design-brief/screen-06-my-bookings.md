Связанный файл: `design-brief.md`
UC: UC-4, UC-6
Порядковый номер: 6

## Цель

Видеть активные и прошедшие записи, управлять ими.

## Связь с требованиями

- FR: FR-35a, FR-46, FR-57
- US: US-12, US-15, US-17
- UC: UC-4, UC-6

## Структура экрана

- Сегмент-контрол: «Предстоящие» / «Прошедшие».
- Список карточек броней, отсортированный по `slot.start_at`.
- Empty state: «У вас пока нет записей» + CTA «Посмотреть расписание».

### Карточка брони

- Дата/время, формат, инструктор.
- Статус-бейдж:
  - `booked` — синий/primary
  - `attended` — зелёный, «Посещено»
  - `cancelled_by_client` — серый, «Отменено вами»
  - `cancelled_late` — жёлтый, «Поздняя отмена»
  - `no_show` — жёлтый, «Неявка»
  - `cancelled_by_gym` — красный, «Отменено скалодромом» + причина
- Снаряжение: иконка + текст.
- Возраст (если детская бронь): «Ребёнок, 8 лет».

### CTA

- `booked` + `can_cancel=true` → «Отменить» (`screen-07-cancel-booking.md`).
- `booked` + `can_cancel=false` → «Написать Оле» (веб-ссылка `https://t.me/...` в Telegram).
- `cancelled_by_gym` → «Выбрать другое время» (открывает расписание с фильтром по формату, FR-46).
- `attended` + нет оценки + прошло ≥ 1ч → «Оценить инструктора» (FR-57) (`screen-08-rate-instructor.md`).

### Баннер для отмен скалодромом (если push не дошёл)

Вверху списка: «N тренировок отменено скалодромом» → раскрытие со списком.

## Состояния экрана

| Состояние | Поведение |
|---|---|
| Загрузка | Skeleton-карточки |
| Пустое | Empty state + CTA «Посмотреть расписание» |
| Данные | Список карточек |
| Offline | Лента из кэша (через Service Worker / Cache API) + баннер «Данные могли устареть» |
| Есть отмены скалодромом | Баннер вверху списка |

## Заметки для дизайнера

- История сортируется по `slot.start_at`.
- Разбиение на «предстоящие» и «прошедшие» — на клиенте.
- В списке отображаются в том числе отменённые брони и поздние отмены (FR-35a).
- Баннер отмен скалодромом — компенсирующий элемент для случая, когда push не дошёл (NFR-28).
- На мобильном — вертикальный список; на десктопе — допустима двухколоночная раскладка (активные / архив).

Экран: Расписание (лента)
Связанный файл: `design-brief.md`
UC: UC-2 (Просмотр и фильтрация списка слотов)
Порядковый номер: 2

Цель
Показать слоты на 7 дней и дать быстро найти нужный.

Связь с требованиями
FR: FR-9, FR-38, FR-52, FR-53
US: US-2, US-3, US-5, US-16
UC: UC-2

Структура экрана
Шапка:
- Заголовок «Расписание».
- Иконка фильтра (справа) → открывает модалку фильтров (`screen-03-filters.md`).
- Активные фильтры — чипы под шапкой (с крестиком для сброса).
Горизонтальный скролл дат (7 дней): «Сегодня / Завтра / Ср / Чт / Пт / Сб / Вс» + кнопка «Ещё» (кастомный диапазон).
Лента слотов — вертикальный список, сгруппированный по дням.
Empty state: «Пока нет доступных тренировок» + кнопка «Сбросить фильтры» (FR-9).

Состояния
| Состояние | Поведение |
|---|---|
| Загрузка | Skeleton-карточки |
| Пустое (E1 UC-2) | Empty state «Пока нет доступных тренировок» + CTA «Сбросить фильтры» |
| Нет расписания (E2 UC-2) | Empty state |
| Данные | Лента карточек |
| Offline (NFR-24) | Лента из кэша + баннер сверху «Данные могли устареть · Обновить». CTA записи недоступны |

Заметки для дизайнера
- Слоты сортируются по времени старта (ближайшие выше).
- Заполненные и отменённые слоты не скрываются — показываются с пометкой (FR-9).
- Горизонтальный скролл дат — удобный паттерн для 7 дней. На десктопе может быть заменён на горизонтальный календарь/сегмент-контроль.
- Активные фильтры отображаются чипами под шапкой для быстрой отмены.
- Карточка слота должна помещать ключевые данные в первый экран: время, формат, свободно мест, цена.

# Экран: Расписание (лента)

> **Связанный файл:** `design-brief.md`  
> **UC:** UC-2 (Просмотр и фильтрация списка слотов)  
> **Порядковый номер:** 2

---

## Цель
Показать слоты на 7 дней и дать быстро найти нужный.

## Связь с требованиями
- **FR:** FR-9, FR-38, FR-52, FR-53
- **US:** US-2, US-3, US-5, US-16
- **UC:** UC-2

---

## Структура экрана

1. **Шапка:**
   - Заголовок «Расписание».
   - Иконка фильтра (справа) → открывает шторку фильтров (`screen-03-filters.md`).
   - Активные фильтры — чипы под шапкой (с крестиком для сброса).
2. **Горизонтальный скролл дат** (7 дней): «Сегодня / Завтра / Ср / Чт / Пт / Сб / Вс» + кнопка «Ещё» (кастомный диапазон).
3. **Лента слотов** — вертикальный список, сгруппированный по дням.
4. **Empty state:** «Пока нет доступных тренировок» + кнопка «Сбросить фильтры» (FR-9).

### Карточка слота в ленте (компактная)
- Время старта (крупно, 24pt, слева).
- Формат: чип «Новички» / «Опытные» (цветовой код).
- Инструктор: имя мелким текстом.
- Справа: `X / N мест` (например, `5 / 8`) + цена `1500 ₽`.
- Индикаторы:
  - 🔴 `Мест нет` (если `free_seats = 0`).
  - ⚠️ `Отменена скалодромом` (если `cancelled_by_gym`).
  - 🔒 `Откроется через N дней` (если слот ещё не открыт для категории клиента, FR-52).

## Состояния экрана

| Состояние | Поведение |
|---|---|
| **Загрузка** | Skeleton-карточки |
| **Пустое** (E1 UC-2) | Empty state «Пока нет доступных тренировок» + CTA «Сбросить фильтры» |
| **Нет расписания** (E2 UC-2) | Empty state |
| **Данные** | Лента карточек |
| **Offline** (NFR-24) | Лента из кэша + баннер сверху «Данные могли устареть · Обновить». CTA записи недоступны |

---

## Состояния карточки слота

| Состояние | CTA | Поведение |
|---|---|---|
| Есть места, слот открыт | «Записаться» (активна) | → карточка слота (`screen-04-slot-detail.md`) |
| Мест нет | «Уведомить, если появится место» | → Alert List (FR-53) |
| Отменена скалодромом | нет CTA, плашка с причиной | → детали с причиной |
| Слот ещё не открыт | неактивна + пояснение «Откроется через N дней» | тап не ведёт в запись |
| Клиент заблокирован | неактивна + «Вы заблокированы до N» | → детали с датой |

---

## Заметки для дизайнера
- Слоты сортируются по времени старта (ближайшие выше).
- Заполненные и отменённые слоты **не скрываются** — показываются с пометкой (FR-9).
- Горизонтальный скролл дат — удобный паттерн для 7 дней.
- Активные фильтры отображаются чипами под шапкой для быстрой отмены.

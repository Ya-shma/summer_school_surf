# LOGIC-010 — Матрица ошибок

## Цель
Единая обработка кодов `4xx` / `5xx`.

## Связь с требованиями
- **NFR:** NFR-25
- **US:** US-25
- **UC:** UC-3, UC-4, UC-5, UC-6

---

## Матрица
| Код | HTTP | UX-реакция | Retry |
|---|---|---|---|
| `slot_full` | 409 | «Места закончились» + обновить карточку | Нет (изменить выбор) |
| `double_booking` | 409 | «Вы уже записаны» → SCR-006 | Нет |
| `rope_access_required` | 403 | «Сначала пройдите вводную тренировку» | Нет |
| `client_blocked` | 403 | «Вы заблокированы до N» | Нет |
| `rental_unavailable` | 409 | «Проката нет, выберите свои скальники» | Нет |
| `slot_cancelled` | 410 | «Тренировка отменена» | Нет |
| `slot_started` | 422 | «Тренировка началась / прошла» | Нет |
| `invalid_code` | 400/401 | «Неверный код, попробуйте ещё раз» | Да |
| `429` + `retry_after` | 429 | «Слишком много попыток, попробуйте через N минут» | Нет (ждать) |
| `5xx` | 5xx | «Что-то пошло на стороне сервера. Повторите через минуту» | Да (с тем же `Idempotency-Key`) |
| Нет сети | — | Баннер «Нет соединения» + Retry | Да |
| Неопределённый результат (таймаут) | — | Retry с тем же `Idempotency-Key` (LOGIC-008) | Да |

---

- Все экраны с мутациями (SCR-001, SCR-004, SCR-007, SCR-008).

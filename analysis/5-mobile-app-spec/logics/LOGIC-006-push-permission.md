# LOGIC-006 — Запрос push-разрешения

## Цель
Один раз предупредить клиента об отключённых push.

## Связь с требованиями
- **FR:** FR-60
- **US:** US-22
- **UC:** UC-8

---

## Входные данные
- Статус разрешения на push на уровне ОС.

## Выходные данные
- Показать / не показать BS-004.

---

## Алгоритм
- При первом открытии приложения, если разрешение на push отклонено на уровне ОС → показать BS-004.
- Показывается **один раз** (флаг хранится локально).
- Не блокирует функционал.
- Кнопка «Открыть настройки» — системный deep-link в настройки уведомлений ОС.

---

## Применяется в
- SCR-002 (Расписание).
- BS-004 (Предупреждение о push).

# LOGIC-009 — Alert List (список ожидания)

## Цель
Подписка на уведомление об освобождении места.

## Связь с требованиями
- **FR:** FR-53, FR-61
- **BR:** BR-6
- **US:** US-16
- **UC:** UC-5

---

## Входные данные
- `slot_id`, действие (подписка / отписка).

## Выходные данные
- Состояние кнопки Alert List.

---

## Алгоритм
1. Клиент нажимает «Уведомить, если появится место» на карточке заполненного слота.
2. Клиент передаёт `Idempotency-Key` (LOGIC-008).
3. Сервер добавляет клиента в Alert List слота.
4. Кнопка меняется на «Вы в списке ожидания» с возможностью отписаться.
5. При освобождении места (отмена другой брони) — сервер отправляет push первому клиенту в Alert List (FR-61).
6. **Уведомление не резервирует место автоматически** — клиент сам записывается через приложение (BR-6).

## Обработка ошибок
| Код | HTTP | UX-реакция |
|---|---|---|
| `waitlist_already_joined` | 409 | Кнопка сразу показывает «Вы в списке ожидания» |
| `slot_not_full` | 409 | Переключить UI на обычную CTA «Записаться» |
| `slot_cancelled` | 410 | Кнопка Alert List скрывается |

LOGIC-003 — Расчёт цены
Цель
Отобразить итоговую стоимость брони.

Связь с требованиями
FR: FR-45
US: US-6
UC: UC-3

Входные данные
`price_total` из ответа API.

Выходные данные
Итоговая цена для отображения.

Алгоритм
Веб-приложение не пересчитывает `price_total` локально.
Использует значение из ответа API (сервер — источник истины).
Модель цены на сервере: `price_total = price × seats_count + rental_price × rental_count`, где `price = 1500 ₽`, `seats_count = 1`, `rental_count ∈ {0, 1}`.
При расхождении локальной оценки и серверного значения приоритет имеет `price_total` из ответа.

Применяется в
SCR-003 (Карточка слота).
SCR-004 (Оформление записи).
SCR-005 (Экран успеха).

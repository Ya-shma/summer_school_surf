## LOGIC-007 — Паттерн состояний экрана
**Цель**
Единый набор состояний для всех экранов.

**вязь с требованиями**
NFR: NFR-24 (Offline)
US: US-23
UC: UC-2, UC-4

**Состояния**
| Состояние | Поведение |
|---|---|
| loading | Skeleton-плейсхолдеры |
| data | Отображение данных |
| empty | Empty state с иллюстрацией + текстом + CTA |
| error | Баннер «Не удалось загрузить» + Retry |
| offline | Чтение из кэша (через Service Worker / Cache API) + баннер «Данные могли устареть · Обновить» |

**Применение**
SCR-002, SCR-003, SCR-006, SCR-007, SCR-009, SCR-010.

Offline (NFR-24)
Чтение — из кэша (через Service Worker / Cache API) с пометкой «данные могли устареть».
Мутации (запись, отмена, оценка, Alert List) офлайн запрещены — показывается понятное сообщение о необходимости подключения.

Применяется во всех экранах с данными из API.

# LOGIC-005 — Фильтрация слотов

## Цель
Отфильтровать список слотов.

## Связь с требованиями
- **FR:** FR-38
- **US:** US-3
- **UC:** UC-2

---

## Входные данные
- Выбранные фильтры: период, формат, инструктор, `only_available`.

## Выходные данные
- Параметры запроса `GET /slots` (`date_from`, `date_to`, `format`, `instructor_id`, `only_available`).

---

## Алгоритм
- **Внутри группы** — OR (например, выбраны и новичковый, и опытный формат → показать оба).
- **Между группами** — AND (например, формат И период дат И инструктор).
- Границы периода дат — включительные.
- Даты вводятся в локальной дате клиента.
- Пресеты: «Сегодня / Эта неделя / Выходные» + кастомный диапазон.
- По умолчанию фильтры не заданы (дефолтная выдача — 7 дней, FR-9).
- Если справочник инструкторов пуст — фильтр по инструктору скрывается (A4 UC-2).

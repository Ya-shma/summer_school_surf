# SCR-008 — Оценка инструктора

## Цель
Быстро оценить после тренировки.

## Связь с требованиями
- **FR:** FR-57, FR-58
- **US:** US-17
- **UC:** UC-6

## Применяемые логики
- LOGIC-007 (Паттерн состояний экрана)
- LOGIC-008 (Идемпотентность мутаций)
- LOGIC-010 (Матрица ошибок)

---

## Структура
1. Заголовок: «Как прошла тренировка?»
2. Инструктор: имя, дата тренировки.
3. 5 звёзд (тап по звезде — выбор).
4. CTA «Отправить оценку» / «Изменить оценку» (активна после выбора).
5. Успех → тост «Спасибо за оценку!» → возврат в SCR-006.

## Состояния экрана
| Состояние | Поведение |
|---|---|
| Нет оценки | 5 пустых звёзд, CTA «Отправить оценку» disabled |
| Выбрана оценка | Звёзды заполнены, CTA активна |
| Уже оценено | Показывает текущую оценку, CTA «Изменить оценку» |
| Загрузка | Loading-индикатор на CTA |
| Успех | Тост «Спасибо за оценку!» |

## Матрица ошибок (LOGIC-010)
| Код | HTTP | UX-реакция |
|---|---|---|
| `not_attended` | 403 | «Оценка недоступна для этой записи» |
| `rating_window_closed` | — | «Окно оценки закрыто» |
| Нет сети | — | Retry с тем же `Idempotency-Key` (LOGIC-008) |

## Ограничения
- Нет текстовых комментариев (FR-58).
- Оценка видна только владельцу.
- Доступно только для статуса `attended` (FR-57).

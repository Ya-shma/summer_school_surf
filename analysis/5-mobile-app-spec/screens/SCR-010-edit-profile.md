# SCR-010 — Редактирование профиля

## Цель
Обновить данные.

## Связь с требованиями
- **FR:** FR-47
- **US:** US-19
- **UC:** UC-7

## Применяемые логики
- LOGIC-007 (Паттерн состояний экрана)
- LOGIC-010 (Матрица ошибок)

---

## Поля
- Имя (обязательно).
- Телефон (только просмотр, смена — через Telegram + ручная привязка владельцем).
- Возраст (опционально).
- День рождения (опционально).

## CTA
- «Сохранить» → при успехе тост «Данные обновлены».

## Состояния экрана (LOGIC-007)
| Состояние | Поведение |
|---|---|
| Загрузка | Skeleton |
| Данные | Поля заполнены |
| Изменения | CTA «Сохранить» активна |
| Успех | Тост «Данные обновлены» |
| Ошибка | Тост «Не удалось сохранить» + Retry |

---

## Навигация
- **Входящая:** SCR-009 («Редактировать данные»).
- **Исходящая:** SCR-009 (успех / back).

# BS-001 — Фильтры слотов

## Цель
Быстро отфильтровать слоты.

## Связь с требованиями
- **FR:** FR-38
- **US:** US-3
- **UC:** UC-2

## Применяемые логики
- LOGIC-005 (Фильтрация слотов)

---

## Структура
1. **Период:** сегмент-контрол «Сегодня / Эта неделя / Выходные / Кастомный» + date-picker для кастомного.
2. **Формат:** чипы «Новички» / «Опытные» (мультивыбор, OR внутри группы).
3. **Инструктор:** список чипов с именами (скрывается, если справочник пуст — A4 UC-2).
4. **Опция:** «Только свободные» (toggle).
5. Кнопки «Сбросить» и «Применить».

## Семантика (FR-38)
- Внутри группы — OR.
- Между группами — AND.

## Состояния
| Состояние | Поведение |
|---|---|
| Пустые фильтры | «Сбросить» disabled |
| Есть активные фильтры | «Сбросить» активна |
| Применение | Закрытие шторки, обновление SCR-002 |

---

## Навигация
- **Входящая:** SCR-002 (тап по иконке фильтра).
- **Исходящая:** SCR-002 (после применения/закрытия).

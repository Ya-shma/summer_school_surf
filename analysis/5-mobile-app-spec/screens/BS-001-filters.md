# BS-001 — Фильтры слотов

## Цель
Быстро отфильтровать слоты.
Связь с требованиями
FR: FR-38
US: US-3
UC: UC-2
Применяемые логики
LOGIC-005 (Фильтрация слотов)
Структура
Период: сегмент-контрол «Сегодня / Эта неделя / Выходные / Кастомный» + date-picker для кастомного.
Формат: чипы «Новички» / «Опытные» (мультивыбор, OR внутри группы).
Инструктор: список чипов с именами (скрывается, если справочник пуст — A4 UC-2).
Опция: «Только свободные» (toggle).
Кнопки «Сбросить» и «Применить».
Семантика (FR-38)
Внутри группы — OR.
Между группами — AND.
Состояния
| Состояние | Поведение |
| ---|---|
| Пустые фильтры | «Сбросить» disabled |
| Есть активные фильтры | «Сбросить» активна |
| Применение | Закрытие модалки, обновление SCR-002 |
Навигация
Входящая: SCR-002 (клик по иконке фильтра).
Исходящая: SCR-002 (после применения/закрытия).
Заметки для дизайнера
Модалка фильтров: на мобильных — bottom sheet (шторка снизу), на десктопе — dialog/dropdown по центру.
Активные фильтры отображаются чипами на основном экране.
Кастомный диапазон дат — date-picker с двумя полями «с / по».

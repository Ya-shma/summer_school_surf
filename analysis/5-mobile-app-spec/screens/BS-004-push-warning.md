BS-004 — Предупреждение о push

Цель
Один раз объяснить клиенту, что он не будет получать напоминания.

Связь с требованиями
FR: FR-60
US: US-22
UC: UC-8

Применяемые логики
LOGIC-006 (Запрос push-разрешения)

Триггер
При первом открытии веб-приложения, если разрешение на push отклонено на уровне браузера.

Структура
Модалка: иконка колокольчика, заголовок «Уведомления отключены».
Текст: «Вы не будете получать напоминания о тренировках. Включите уведомления в настройках браузера.»
Кнопки: «Открыть настройки» (обычная веб-ссылка на настройки уведомлений браузера) / «Понятно» (закрыть).
Показывается один раз, не блокирует функционал.

Состояния
| Состояние | Поведение |
|---|---|
| Push разрешён | Модалка не показывается |
| Push запрещён (первый раз) | Показывается модалка |
| Push запрещён (повторно) | Модалка не показывается |

Навигация
Входящая: SCR-002 (при первом запуске с отклонённым push).
Исходящая: SCR-002 (после закрытия); настройки уведомлений браузера (веб-ссылка).

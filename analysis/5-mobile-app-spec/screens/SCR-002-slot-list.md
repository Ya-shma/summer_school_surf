# SCR-002 — Расписание (лента слотов)

## Цель
Показать слоты на 7 дней и дать быстро найти нужный.

## Связь с требованиями
- **FR:** FR-9, FR-38, FR-52, FR-53
- **US:** US-2, US-3, US-5, US-16
- **UC:** UC-2

## Применяемые логики
- LOGIC-002 (Расчёт доступности слота)
- LOGIC-005 (Фильтрация слотов)
- LOGIC-006 (Запрос push-разрешения)
- LOGIC-007 (Паттерн состояний экрана)

---

## Структура
1. **Шапка:**
   - Заголовок «Расписание».
   - Иконка фильтра → BS-001.
   - Активные фильтры — чипы под шапкой (с крестиком для сброса).
2. **Горизонтальный скролл дат** (7 дней): «Сегодня / Завтра / Ср / Чт / Пт / Сб / Вс» + кнопка «Ещё» (кастомный диапазон).
3. **Лента слотов** — вертикальный список, сгруппированный по дням.
4. **Empty state:** «Пока нет доступных тренировок» + CTA «Сбросить фильтры» (FR-9).

### Карточка слота в ленте (компактная)
- Время старта (крупно, 24pt, слева).
- Формат: чип «Новички» / «Опытные» (цветовой код).
- Инструктор: имя мелким текстом.
- Справа: `X / N мест` + цена `1500 ₽`.
- Индикаторы: 🔴 `Мест нет`, ⚠️ `Отменена скалодромом`, 🔒 `Откроется через N дней`.

## Состояния карточки (LOGIC-002)
| Состояние | CTA | Поведение |
|---|---|---|
| Есть места, слот открыт | «Записаться» (активна) | → SCR-003 |
| Мест нет | «Уведомить, если появится место» | → Alert List (LOGIC-009) |
| Отменена скалодромом | нет CTA, плашка с причиной | → детали с причиной |
| Слот ещё не открыт | неактивна + пояснение | тап не ведёт в запись |
| Клиент заблокирован | неактивна + «Вы заблокированы до N» | → детали с датой |

## Состояния экрана (LOGIC-007)
| Состояние | Поведение |
|---|---|
| Загрузка | Skeleton-карточки |
| Пустое (E1 UC-2) | Empty state + CTA «Сбросить фильтры» |
| Данные | Лента карточек |
| Offline (NFR-24) | Лента из кэша + баннер «Данные могли устареть · Обновить» |
